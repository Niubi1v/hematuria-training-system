# 血尿病例医学事实专家审核说明

## 当前口径

- 42 例病例均保持 `needs_revision`，不可用于正式 OSCE 或 RCT。
- 572 条事实追踪项中，153 条可追溯到原始病例资料。
- 其中 2 条旧追踪记录已与运行时来源重新对齐：`P003.transfusionHistory`、`P005.coronaryDisease`。
- 其余 419 条属于为标准化患者连续问答而补充的模拟事实，必须由专家逐条裁决。

## 审核角色

- 泌尿外科：肿瘤、结石、感染、前列腺、外伤及泌尿操作相关事实。
- 肾内科/儿童肾内科：肾小球疾病、遗传性肾病及儿童病例。
- 妇产科：月经、妊娠和阴道污染相关事实。
- 心内科/血液科/神经内科：抗栓药、冠心病、卒中和围术期停药风险。
- 医学教育：患者口吻、问答粒度、评分相关性和教学可接受性。

## 每条事实的必要签署字段

1. 决定：保留、修改或删除。
2. 修改后的中文与英文患者回答；选择修改时两项均必填。
3. 病例依据或指南依据。
4. 审核人姓名、专科和日期。
5. 必要时填写审核备注。

未完成上述字段的事实必须保持 `待确认`，不得由程序自动改为 approved。

## 建议顺序

1. 先处理 `P0-安全相关`：抗凝/抗血小板、过敏、妊娠、冠心病、卒中和手术史。
2. 再处理 `P1-诊断评分相关`：吸烟、职业暴露、结石/感染/肿瘤史、月经和家族史。
3. 最后处理 `P2-背景完整性`：饮酒、一般慢病、输血及其他背景信息。

审核完成的工作簿应由项目维护者导入并重新运行医学矛盾检查、双语测试、病例回归和静态泄露扫描。正式模式解锁仍需病例级医学负责人签署，不能仅以事实行全部填写作为自动批准。
