# ICP、域名与公网开放人工清单

此清单不构成法律意见，也不会自动购买、备案或修改 DNS。备案完成前，不将中国内地 POC 对公网正式开放。

## 主体和资源

- [ ] 确认备案主体（单位优先）和负责人。
- [ ] 完成中国大陆云账号实名认证。
- [ ] 购买中国内地地域、满足接入商备案资格和时长的服务器。
- [ ] 域名注册商资质、后缀和有效期符合备案要求。
- [ ] 域名完成实名认证，持有人信息与备案主体匹配。
- [ ] 确认既有备案是否需要“新增接入”或“新增网站”。
- [ ] 由备案顾问/法务判断临床教育内容是否涉及医疗、药品、出版等前置审批。

## ICP 备案

- [ ] 在实际服务器接入商（腾讯云或阿里云）提交，不跨厂商代备案。
- [ ] 准备主体证件、负责人信息、网站名称、服务内容和真实性核验材料。
- [ ] 完成接入商初审、短信核验和管局审核。
- [ ] 获得备案号后在页面底部展示并链接工信部备案系统。
- [ ] 网站开放后按适用要求完成公安联网备案。

腾讯云明确中国内地轻量服务需要备案：[使用限制](https://cloud.tencent.com/document/product/1207/44376/)。阿里云备案流程说明中国内地服务器须在所属接入商办理，并提示部分行业可能需要前置审批：[备案流程](https://help.aliyun.com/zh/icp-filing/basic-icp-service/user-guide/icp-filing-application-overview)。

## 域名和证书

- [ ] 使用独立审阅子域名，例如 `review.example.cn`。
- [ ] 备案完成后才把 A/AAAA 记录指向大陆服务器。
- [ ] DNS TTL 在切换前合理降低；记录旧值和回退步骤。
- [ ] 申请与子域名完全匹配的 HTTPS 证书。
- [ ] 私钥仅在服务器保存，权限最小，不进入 Git/镜像/日志。
- [ ] 验证完整证书链、TLS 1.2/1.3、证书自动续期和到期告警。
- [ ] 确认 HTTP 仅用于 ACME/健康并跳转 HTTPS。

## 网络

- [ ] 公众仅开放 80/443。
- [ ] SSH 仅管理员固定 IP，禁密码登录并启用密钥。
- [ ] Redis 仅私网，无公网端点，无宿主机端口映射。
- [ ] Redis 白名单最小化，VPC 网段不冲突。
- [ ] Origin allowlist 只含正式 HTTPS 域名，不含 `*`。
- [ ] 备案完成前使用本机 Compose、内网或经批准的受限访问方式验证；不以用户电脑作为公网服务器。

## 开放审批

- [ ] 医学内容负责人确认预发布仅作训练/审阅用途。
- [ ] 安全负责人完成清单和 secret 扫描。
- [ ] 运维负责人完成备份、恢复、告警和回滚演练。
- [ ] 产品负责人确认 mock 结果明确标记，不当作真实 AI 验收。
- [ ] 备案、域名、证书、性能和教师验收证据归档。
