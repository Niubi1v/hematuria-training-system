# 域名、备案与 HTTPS 清单

- [ ] 域名主体、云账号实名主体和备案主体关系已由负责人确认。
- [ ] 预发布域名与正式域名分离。
- [ ] 在腾讯云按实际情形完成人工 ICP 首次备案或接入备案。
- [ ] 备案完成前不将大陆 CVM 作为对公众开放的 Web 服务。
- [ ] 公安备案、隐私政策、用户告知等要求由合规负责人判断并执行。
- [ ] DNS 解析由用户在控制台人工修改，有变更记录和回退 TTL 计划。
- [ ] HTTPS 证书覆盖确切域名，私钥只在服务器，TLS 1.2/1.3 生效。
- [ ] 配置证书续期与 30/14/7 天到期告警。
- [ ] 验证 HTTP 到 HTTPS 跳转、证书链、HSTS 和同源 API。

本仓库不提供法律结论，也不自动申请备案、购买域名、修改 DNS 或签发证书。
