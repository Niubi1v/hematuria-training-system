版权边界：未复制人卫版《诊断学》或《中国泌尿外科和男科疾病诊断治疗指南》原文。若你持有正版PDF，可将其作为更高优先级RAG源接入；本文件用于训练系统原型和教学路径设计。 一、血尿教学总框架 血尿训练系统的第一目标不是让学生直接猜诊断，而是训练其按顺序完成：确认真性血尿、判断严重程度、定位内科性/外科性来源、识别急症、选择检查、发起MDT、制定治疗与随访。 先确认：尿液变红不等于血尿。尿潜血阳性需要尿沉渣镜检确认红细胞；若潜血阳性但镜检红细胞不匹配，要考虑血红蛋白尿、肌红蛋白尿或假性血尿。 再分类：肉眼血尿/镜下血尿；起始/终末/全程血尿；有无血块；疼痛性/无痛性；间断/持续；感染/结石/肿瘤/肾小球线索。 再分层：高龄、男性、吸烟、职业暴露、无痛肉眼血尿、血块、反复发作属于泌尿系肿瘤高危线索。 同时保留肾内科安全网：蛋白尿、泡沫尿、水肿、高血压、畸形红细胞、红细胞管型、AKI、上感后血尿提示肾小球性血尿。 最后再治疗：肿瘤治疗必须基于病理与分期；感染梗阻结石先引流抗感染，不能直接碎石；抗凝药不能作为血尿唯一解释。 二、诊断学症状学：问诊必须覆盖的血尿要点 模拟患者系统必须做到“学生问到哪个点，就能返回明确答案”。尤其是血尿时相：学生问“是不是一直红”“是不是拉到最后才红”“刚开始红还是最后红”，系统都要能按病例回答。 问诊点 模拟患者答案类型 教学意义 血尿可见性 肉眼能见红/洗肉水样/茶色，或体检尿检发现镜下血尿 决定风险分层和后续检查强度。肉眼血尿通常更需要肿瘤排查。 血尿时相 起始、终末、全程、镜下、无法判断 起始偏尿道；终末偏膀胱颈/三角区/后尿道/前列腺；全程偏膀胱以上或出血量较大。只是线索，不是最终诊断。 是否一直红 从开始到结束都红，还是只在末端/起始出现 网页必须给出明确回答，例如“不是一直红，是快尿完时更明显”。 颜色 鲜红、暗红、洗肉水样、茶色/可乐色、粉红 鲜红和血块偏下尿路/非肾小球；茶色/可乐色结合蛋白尿和管型提示肾小球来源。 血块 无、有小血块、有条索状血块、有血块堵塞尿道 血块常提示非肾小球来源；血块尿潴留是急症。 疼痛 无痛、尿痛、肾绞痛、腰痛、下腹痛 无痛肉眼血尿警惕肿瘤；肾绞痛警惕结石；尿痛尿急提示感染。 感染症状 尿频、尿急、尿痛、发热、寒战、肾区痛 区分局限性UTI与系统性UTI；系统性症状需血液检查、影像、静脉抗感染和住院评估。 肾小球线索 泡沫尿、水肿、高血压、上感后、皮疹关节痛、听力异常 触发肾内科会诊和尿蛋白/肾功能/免疫学/肾活检评估。 危险因素 吸烟、职业暴露、放疗、环磷酰胺、结石史、反复感染、家族史 训练低年级住培识别肿瘤、结石、肾病和药物相关风险。 假性血尿 月经/阴道出血、色素食物、药物、血红蛋白尿/肌红蛋白尿 女性病例必须排除污染；红色尿要结合尿沉渣。 三、检查检验开单路径 系统设计上，住培医生必须输入具体医嘱，Order Result Agent 才返回对应结果；不能在病例一开始就把所有检查结果暴露。 路径 推荐开单 系统返回/评分意义 基础必开 尿常规+尿沉渣、血常规、肾功能/eGFR、电解质、凝血功能 确认真性血尿、贫血/感染、造影与用药安全、围术期风险。 感染路径 尿培养+药敏、血培养、CRP/PCT/乳酸、泌尿系超声或CT 发热寒战/系统性UTI/梗阻感染时必须采样并尽早抗感染。 结石路径 非增强CT KUB、尿pH、结石成分分析、代谢评估 急性肾绞痛定位结石大小位置；复发结石需要预防策略。 肿瘤路径 CTU、膀胱镜、尿细胞学、TURBT病理、增强CT/MRI分期 高危肉眼血尿不能只做超声；肿瘤治疗以病理和分期为依据。 肾小球路径 尿蛋白/肌酐比或24h尿蛋白、尿RBC形态、肾功能、补体、ANA/ANCA/抗GBM等、必要时肾活检 蛋白尿/管型/畸形RBC/AKI时必须触发。 女性/妊娠 清洁中段尿或导尿尿标本、妊娠试验、超声优先 排除污染，避免不必要放射和禁忌用药。 外伤路径 增强CT、CT膀胱造影、逆行尿道造影 外伤后肉眼血尿/骨盆骨折/尿道口出血需排查泌尿系损伤。 四、疾病路径与治疗要点 疾病路径 关键线索 检查策略 治疗/会诊要点 膀胱癌 高龄男性、无痛性肉眼血尿、血块、吸烟/职业暴露 尿常规、肾功能、泌尿系影像、膀胱镜；影像明确肿瘤可TURBT TURBT病理和分期后决定NMIBC/MIBC路径；不能无病理治疗。 UTUC 肉眼血尿、腰痛/血块梗阻、肾盂/输尿管占位 CTU/MRU、膀胱镜、尿细胞学、必要时输尿管镜活检 按风险分层考虑肾输尿管切除或保肾治疗；高危/转移需MDT。 结石 突发肾绞痛、向腹股沟放射、恶心呕吐、镜下/肉眼血尿 尿常规、血常规、肌酐、CRP、超声、非增强CT 镇痛、补液、止吐；感染/梗阻/AKI则急诊处理。 感染性梗阻 发热寒战、肾区痛、梗阻结石/肾积水、无尿/AKI 尿血培养、血常规、CRP/PCT、影像确认梗阻 立即抗感染+输尿管支架或肾造瘘引流，感染控制后再碎石。 局限性UTI/系统性UTI 尿频尿急尿痛；若发热寒战/低血压/肾区痛为系统性 尿常规、尿培养；系统性需血液检查和影像 按严重程度选择门诊口服或住院静脉抗菌药，治疗后复查血尿。 肾小球性血尿 茶色/可乐色、泡沫尿、水肿、高血压、上感后、蛋白尿/管型 尿蛋白定量、肾功能、免疫学、肾内科会诊、必要时肾活检 控制血压和蛋白尿，具体免疫治疗由肾内科根据病理决定。 BPH/前列腺相关 老年男性、LUTS、终末血尿/尿潴留 尿常规、残余尿、前列腺体积、PSA解释、必要时膀胱镜 解除尿潴留、药物/手术；但不能漏排膀胱癌。 抗凝相关血尿 长期阿司匹林/氯吡格雷/华法林/DOAC 同等完成血尿病因评估；术前凝血和心脑血管风险评估 抗凝可加重血尿但不是唯一病因；围术期MDT。 五、MDT触发机制 本项目不是单纯诊断训练，还要训练低年级住培医生“知道什么时候不能单科处理”。以下触发点应进入Evaluator核心评分。 会诊科室 触发条件 会诊要解决的问题 肾内科 蛋白尿、RBC管型/畸形RBC、AKI、显著高血压、水肿、泡沫尿、上感后血尿 判断肾小球性血尿；是否需肾活检；控制血压/蛋白尿；是否免疫治疗。 感染科/ICU 系统性UTI、发热寒战、低血压、心动过速、耐药菌、免疫抑制、感染性梗阻 抗菌药选择、培养解读、脓毒症处理、降阶梯治疗。 影像科 CTU不明确、肾占位分期、输尿管充盈缺损、肾静脉/下腔静脉瘤栓、外伤损伤分级 选择合适影像；解释病变来源和分期；指导介入或手术规划。 泌尿肿瘤/肿瘤内科 MIBC、高危UTUC、转移性尿路上皮癌、肾癌疑转移、需要新辅助/辅助治疗 分期、病理、全身治疗、MDT治疗顺序。 血管外科 巨大肾占位伴肾静脉/下腔静脉瘤栓、复杂血管受累 瘤栓范围、血管控制、联合手术和风险评估。 麻醉科/ICU 严重贫血、感染休克、重大手术、肾功能不全、高龄衰弱、抗栓药 围术期风险评估、输血、术后ICU、抗栓管理。 心内科/神经内科/血液科 冠脉支架、房颤、脑梗、机械瓣、长期抗凝/抗血小板、凝血异常 停药/桥接/继续抗栓方案，平衡血栓与出血风险。 妇产科 女性血尿样表现与月经、阴道出血、妊娠混杂 排除妇科出血污染；妊娠期影像和用药安全。 病理科 TURBT、肾穿刺、输尿管镜活检、肾肿瘤术后病理 确认组织学、分级、肌层是否存在、免疫组化/分子检测。 创伤外科/介入科 外伤后肉眼血尿、休克、肾裂伤、盆骨骨折、疑膀胱/尿道损伤 出血控制、损伤分级、介入栓塞或手术。 六、多智能体护栏 Agent 角色 允许行为/提示词要点 禁止行为 Patient Agent 模拟患者 只回答医生问到的内容；不主动泄露吸烟、抗凝药、职业暴露、家族史、上感等关键线索。回答要符合病例年龄、情绪和疾病。 不能给诊断、不能解释检查意义、不能说出教师端答案。 Order Result Agent 检查检验返回 仅当医生明确下达医嘱后返回对应结果；若医嘱含糊，应要求明确项目。 不能一次性吐出全部检查；不能给未开项目结果。 MDT Consultants Agent 多学科专家 根据被邀请科室身份发言：肾内科/影像/感染/肿瘤/麻醉/血管外科等；先确认关键信息是否足够，再给建议。 不能替代主治医生做最终决定；不能超出本科室专业边界。 Evaluator Agent 后台导师评分 隐蔽运行，记录问诊槽位、开单、诊断、MDT、治疗与随访；按rubric实时扣分。 不能提前暴露标准答案；不能与学生直接讨论隐藏评分细节。 Orchestrator Agent 主控调度 管理训练阶段、会话状态、可见字段、已问槽位、已开医嘱、MDT开启状态。 不得让前端直接读隐藏答案。 七、Evaluator扣分规则摘要 扣分主题 错误表现 系统反馈 未确认真性血尿 未开尿常规/尿沉渣就诊断 严重扣分；提示缺乏诊断依据。 漏问血尿时相 未问起始/终末/全程或是否一直红 问诊扣分；无法完成症状学定位。 漏问肿瘤风险 无痛肉眼血尿病例未问吸烟、职业暴露、血块、体重下降 高危漏项，触发肿瘤安全网。 漏掉肾内科线索 蛋白尿/泡沫尿/水肿/高血压/上感后未问未查 触发肾内科专家质询。 感染梗阻误处理 发热梗阻直接碎石或只口服抗生素 危险错误，重扣分。 抗凝归因错误 将阿司匹林/华法林当作唯一原因，不继续排查 高危错误。 不及时MDT 巨大肾占位伴瘤栓/严重贫血/晚期肿瘤不发起MDT 核心胜任力扣分。 八、建议放入项目的RAG目录结构 rag_sources/
