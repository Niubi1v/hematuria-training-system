1. 读取 xlsx 中以下工作表：
   - 知识条目_路径规则
   - 问诊槽位_诊断学
   - 开单返回_项目库

二、知识优先级
本院路径/带教老师审核规则 > 中国指南正版PDF > 人卫《诊断学》教材 > EAU/AUA指南 > NCBI/政府医学资料 > 其他综述。若来源冲突，以优先级高者为准，并在教师端提示“需人工确认”。

三、信息分层释放
1. 问诊阶段：Patient Agent只返回被问到的槽位答案，不主动透露危险因素。
2. 开单阶段：Order Result Agent只返回已开医嘱的检查结果。
3. 诊断阶段：若未完成尿常规/尿沉渣确认真性血尿，提示“缺乏诊断依据”。
4. 治疗阶段：无病理/分期不得给肿瘤确定性治疗；感染性梗阻不得直接碎石。

四、必须实现的安全网
1. 高危血尿安全网：高龄、男性、吸烟、职业暴露、无痛肉眼血尿、血块、反复发作，必须触发膀胱癌/UTUC/RCC排查。
2. 肾内科安全网：蛋白尿、泡沫尿、水肿、高血压、RBC管型/畸形RBC、AKI、上感后血尿，必须触发肾小球性血尿鉴别和肾内科会诊提示。
3. 感染梗阻急症：发热寒战/脓毒症 + 结石/肾积水/梗阻/无尿，必须提示抗感染、培养、急诊引流，禁止直接碎石。
4. 抗凝药规则：阿司匹林、氯吡格雷、华法林、DOAC不能作为血尿唯一解释。
5. 女性污染规则：女性血尿样表现必须排除月经/阴道出血污染和妊娠。

五、自由问诊匹配

- “是不是一直红？” -> HX006
- “是拉到最后才红吗？” -> HX007
- “刚开始红还是最后红？” -> HX005/HX008/HX007
- “有没有血块？” -> HX010
- “像茶色还是鲜红？” -> HX003
- “有没有泡沫尿/水肿/高血压/上感？” -> HX019-HX022
- “有没有抽烟/职业暴露/抗凝药？” -> HX027-HX030

六、后台Evaluator评分
记录：
- 已问槽位/未问关键槽位
- 已开医嘱/漏开关键项目/不必要项目
- 是否识别急症
- 是否完成内科性 vs 外科性血尿鉴别
- 是否按时发起MDT
- 是否存在高危错误
最终输出：总分、维度分、漏项、危险错误、复习建议。

七、教师端
教师端显示每条规则的来源source_id、RAG片段、路径解释和人工覆盖按钮。学生端不得显示最终诊断和评分答案。
